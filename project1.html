Network Security Lab with pfSense and VLANs

Overview

In this project, I set up a lab environment with pfSense as the firewall, VLANs, and various security tools like Wazuh and Kali Linux.

Objectives

  • Configure pfSense with multiple VLANs (1, 10, 20, 30).
  • Isolate devices in different VLANs for security testing.
  • Install security tools in different VLANs, like Wazuh, Kali, and Docker containers (DVWA, WebGoat).

Network Setup

Hyper-V Configuration

  • WAN and LAN setup with pfSense as the virtual firewall.
  • pfSense network IDs: VLAN 1 (LAN), VLAN 10, VLAN 20, VLAN 30.
  • Physical connection to the second laptop for VLAN 1 (LAN).
